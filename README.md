# portal-main-notes

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面通常是静态的、功能自包含的，不针对任何特定域名或网站，旨在作为笔记、工具或演示页面的集合。

## 目录说明

```
portal-main-notes/
├── pages/            # 存放所有 HTML 页面文件
├── assets/           # 公共资源（CSS、JS、图片等）
├── docs/             # 文档或说明文件
└── README.md         # 本文件
```

- **pages/**：每个子目录或文件对应一个独立的 HTML 页面，命名清晰，便于识别。
- **assets/**：页面间可能共享的样式、脚本或媒体资源。
- **docs/**：包含使用说明、版本记录或其他补充文档。

## 页面归档说明

- 所有页面均为纯前端 HTML，不依赖后端服务。
- 每个页面在 `pages/` 下以独立文件或文件夹形式存放，内容自解释。
- 页面之间彼此独立，无交叉引用或依赖关系。
- 归档的页面可能包含实验性、个人笔记或临时工具，不保证长期可用或更新。

## 维护说明

- 本仓库由个人或团队维护，更新频率不固定。
- 欢迎通过 Issue 报告问题或提出建议，但请注意本仓库不针对具体业务或服务。
- 拉取请求（Pull Request）需确保新增页面不影响现有文件结构。
- 资源文件请尽量放置在 `assets/` 目录下，避免分散在多个页面文件夹中。

## 其他

- 本仓库内容仅供学习、参考或存档用途。
- 如有特殊使用需求，请参考各页面内的注释或说明。
